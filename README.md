# MindSpark (InterviewAI) 面之光

**基于大语言模型的沉浸式全真模拟面试与评测平台**

---

### 🚀 项目简介
针对互联网招聘寒冬下应届生“缺乏真实高压面试环境”的痛点，我独立负责研发了这款智能面试辅助平台。它不仅是题库，更是具备“连环施压追问”与“即时逻辑诊断”功能的 24 小时 AI 私教。

### ✨ 核心特性
* **沉浸式对线体验：** AI 基于面试官人设进行深度追问，还原大厂面试高压场景。
* **智能复盘报告：** 自动提取技术栈盲区，通过 Echarts 生成多维能力雷达图。
* **异步 AI 引擎：** 使用 Spring WebClient 实现非阻塞调用，确保高并发下的交互流畅度。

### 🛠️ 技术架构
* **后端：** Java 17 + Spring Boot 3.x + Spring Security
* **数据库：** MySQL 8.0 + Spring Data JPA
* **核心组件：** WebClient 异步通信、JSON 动态解析、RBAC 权限模型
* **部署：** Maven 模块化构建
